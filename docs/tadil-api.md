# Tadil Admin API

## Overview
`tadil-api` is the NestJS-based backend application responsible for serving the Admin Dashboard (`tadil-admin-web-client`).

## Responsibilities
- Managing users (tailors, couriers, customers).
- Managing models, sections, alterations, extras, and informations.
- Handling login request approvals and rejections for tailors and couriers.
- Monitoring all orders with status, tailor, and courier filtering.
- Managing manual tailor assignments for pending orders.
- Fulfilling and rejecting payout requests from service providers.
- Providing data to the admin web client for overall platform management.
