# Tadil Admin Web Client

## Overview
`tadil-admin-web-client` is a Vue 3-based administrative dashboard for managing the Tadil platform.

## Responsibilities
- Managing the catalog of models, sections, alterations, and extras.
- Managing user roles (customers, tailors, couriers).
- Approving or rejecting tailor/courier login requests.
- Viewing platform-wide metrics and orders.

## Tech Stack
- Vue 3 (Composition API)
- Vite
- Tailwind CSS v4
- Vue Router & Vue I18n
